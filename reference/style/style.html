theme(style)

This is an optional function that makes your workflow much faster. This style parameter takes in an object of
primary color (p)
secondary color (s)
text color (c)
font type (font)
border color (border)
And makes a theme out of these and apply it to the html

var style={
p:"#212121",
s:"#424242",
c:"white",
font:"monospace",
border:"white"
}
theme(style);

This is meant for fast purposes

applyStyle(element,style)

This is a fast way of applying styles or class or text or onclick etc to an element

var elm=element("#hello");
var style={
backgroundColor:"red",
color:"white",
text:"hello",
onclick:function(){
alert("hello");
}
}
applyStyle(elm,style)
/*This applys the style as specified in the style object to the element of id hello*/

Not all css styles are added to it only the important ones
