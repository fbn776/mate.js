<Utility

element(name)

This function returns the element that is specified in the bracket. To choose a. element use the element's CSS selector

var elm;
elm=document.getElementsByClassName(".container");
/* You can use this instead of the standard one */ elm=element(".container");

This function only returns a single element

elements(name,n)

This function returns the element from a group of elements with the specified index number

var elm=elements("div",4);
//This returns the 5th div element

Index number starts from 0.

log(item)

This is similar console.log() but shortened

console.log("Hello");
/*This does the same work*/
log("hello");

When this function is called the line number is shown as that of the library

animate(element,steps)

This is a function that enables the css like animation using JavaScript
The function has two parameter:

element:This is the element that is given the animation
steps:This is similar to keyframes in css.

var elm=element(".animate");
var animation={
timing:1000,
delay:100,
from:{
width:"100px"
},
to:{
width:"200px"
}
}
var key=new animate(elm,animation);
key.start();
/*This runs the animation*/
key.reset();
/*This reset the animation*/

This function depends opon the applyStyle() function
